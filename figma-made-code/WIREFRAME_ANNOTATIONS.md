## Component Specifications

### Artwork Card
- **Dimensions**: Flexible width, 3:4 aspect ratio
- **Image**: Object-cover, rounded corners (8px)
- **Badges**: 12px padding, 12px font, top-left absolute
- **Title**: 18px Playfair Display
- **Artist**: 14px Inter, gray-600
- **Price**: 16px Inter, indigo color
- **Hover**: Scale image 110%, 500ms transition

### Button Sizes
- **Small**: 32px height, 12px padding
- **Medium**: 40px height, 16px padding
- **Large**: 48px height, 24px horizontal padding

### Form Fields
- **Height**: 48px (mobile), 56px (desktop)
- **Padding**: 16px horizontal
- **Border**: 1px gray-300
- **Focus**: 2px indigo ring
- **Font**: 16px Inter (prevents zoom on mobile)

### Navigation
- **Height**: 64px
- **Logo**: 24px Playfair Display
- **Links**: 16px Inter, 32px horizontal spacing
- **Cart badge**: 20px diameter, coral background

### Footer
- **Background**: Indigo (#3730A3)
- **Text**: White, 14px Inter
- **Headings**: 16px Inter semi-bold
- **Link hover**: Coral color
- **Padding**: 48px vertical, responsive horizontal

## Color Usage Guide

### When to use Brand Indigo (#3730A3)
- Primary navigation elements
- Primary buttons (secondary CTA)
- Footer background
- Links hover states
- Price displays
- Focus rings

### When to use Accent Coral (#FF6B6B)
- Primary CTA buttons
- Cart badge
- "New" badges
- Active states
- Hover accents
- Important callouts

### Gray Scale
- **50-100**: Backgrounds, subtle sections
- **200-300**: Borders, dividers
- **400-500**: Placeholders, disabled states
- **600-700**: Body text, secondary text
- **800-900**: Headings, emphasis
